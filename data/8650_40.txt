   À l'heure actuelle, sur toutes les box et logiciels testés, il n'est
   pas encore possible de commander de façon différenciée la fonction
   carillon et la fonction sirène (un seul actionneur est proposé). Il
   sera donc nécessaire de choisir une sonorité (de type carillon ou
   sirène) et un niveau sonore par défaut et qui sera celui déclenché.
   *^1 sur eedomus : Lors de notre dernier test du 31/01/2019, seule la
   fonction sirène pouvait être déclenchée.
   *^2  Sur Jeedom : Pour le déclenchement carillon/sirène, il est
   nécessaire de créer manuellement les commandes suivantes :
   - Nom : ON / Type : Action / Instance : 1 / Classe : 37 / Index : 0 /
   Commande : type=setvalue&value=99
   - Nom : OFF / Type : Action / Instance : 1 / Classe : 37 / Index : 0 /
   Commande : type=setvalue&value=0
   *^3  Sur Vera : Pour que la sirène soit complètement compatible, il
   faut modifier dans la configuration du module (onglet Avancé) :
   - type d'appareil : urn:schemas-upnp-org:device:Siren:1
   - device_file : D_Siren1.xml
   - category_num : 24
   - device_json : D_Siren1.json
