Référentiel Général d'Accessibilité pour les Administrations (RGAA) Version 3
2017

Contenus du RGAA 3

   Tous les documents composant le RGAA 3 sont juridiquement opposables.
   Une section ressources, non normative, complète le dispositif.

   Les documents composant le RGAA sont les suivants :
     * Le document « Introduction au RGAA », qui définit la problématique
       et les enjeux de l'accessibilité numérique ;
     * Le « Guide d'accompagnement » ;
     * Le « Référentiel technique », lui-même composé de 6 parties :
          + La liste des critères et tests ;
          + Le glossaire ;
          + Les cas particuliers ;
          + Les notes techniques ;
          + La base de référence ;
          + Les références ayant conduit à l'élaboration du référentiel
            technique.

Versions du RGAA (depuis 2009)

   Le RGAA est un document évolutif, qu'il est nécessaire de réactualiser
   régulièrement pour tenir compte des évolutions normatives et
   techniques. Cette section permet d'accéder aux différentes versions du
   RGAA.

Version courante (RGAA 3 2017)

     * Introduction au RGAA
     * Guide d'accompagnement
     * Référentiel technique

   Chacun des 3 documents composant le RGAA 3 2017 est disponible aux
   formats PDF et ODT :
     * Introduction au RGAA : format PDF (145 kilo-octets), format ODT
       (69,4 kilo-octets) ;
     * Guide d'accompagnement : format PDF (333 kilo-octets), format ODT
       (99 kilo-octets) ;
     * Référentiel technique : format PDF (1,3 méga-octets), format ODT
       (163 kilo-octets).
